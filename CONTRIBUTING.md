# Contributing

Thank you for your interest in contributing to the ORION project.

## How to Contribute

1. **Report Issues**: Use GitHub Issues for bugs, questions, or feature requests
2. **Submit Pull Requests**: Fork the repo, make changes, submit a PR with clear description
3. **Improve Documentation**: Help clarify or expand existing documentation
4. **Add Tests**: Contribute test cases that improve coverage and reliability

## Code Standards

- Python 3.10+ with type hints
- Docstrings for all public functions and classes
- Follow existing code style and patterns
- No external dependencies without discussion

## Process

1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Push to the branch
5. Open a Pull Request

## Questions?

Open an issue or start a discussion.
